1. Introducción

El objetivo de este taller es mostrar de una manera prática la utilización de git, mediante desarrollo de un proyecto real: Academia Template

2. Metodología

El taller esta diseñado para realizar equipos de 3 personas, cada uno de ellos debe seleccionar un rol que corresponde a cada pestaña del historial de tareas

Una vez seleccionado el rol debe seguir los pasos que estan en esa pestaña, pero debe tener en cuenta que algunos pasos dependen de los otros compañeros del equipo

3. Instrucciones

Para poder realizar este taller cada miembro del equipo debe cumplir los siguientes requisitos:

  1. Seleccionar un rol
    Front End Web Developer, UX Designer, Front End Web Designer
  2. Tener una cuenta activa de github
  3. Instalar git en el equipo de trabajo
  4. Configurar el nombre y correo (correspondiente a la cuenta de github) en git
  5. Abrir la terminal y cambiarse al directorio del usuario (~)
