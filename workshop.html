2

  1. Crear un directorio vacio
    mkdir academia[numero-del-grupo]
  2. Instalar Foundation
    CRTL+SHIFT+P Fetch - Package Foundation
  3. Inicializar el repositorio git
    git init
  4. Añadir todos los archivos al stage (index)
    git add .
  5. Realizar el primer commit
    git commit -m "Initial Commit"
  6. Añadir el repositorio remoto al repositorio local
    git remote add origin https://github.com/[nombre-de-usuario]/academia[numero-del-grupo].git
  7. Publicar los cambios en el repositorio remoto
    git push -u origin master
